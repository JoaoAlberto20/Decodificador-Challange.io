# Decodificador-Challange

Essa aplicação é uma criptografa textos, assim você poderá trocar mensagens secretas com outras pessoas que saibam o segredo da criptografia utilizada.

As "chaves" de criptografia que utilizei são:

A letra "e" é convertida para "enter"

A letra "i" é convertida para "imes"

A letra "a" é convertida para "ai"

A letra "o" é convertida para "ober"

A letra "u" é convertida para "ufat"
